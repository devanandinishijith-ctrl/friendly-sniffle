Tkinter Login Portal

This is a simple **Python Tkinter mini-project** that demonstrates a basic login portal with a graphical user interface (GUI).  
It accepts a username and password, and displays a message based on whether the login is successful or not.

---

Features

- Clean and simple GUI using Tkinter
- Username and password input fields
- Password masked with `*`
- Login validation with popup messages
- Error handling with message boxes

---
How It Works

The program checks if:
- **Username:** `Devanandini`  
- **Password:** `123`

If both match, it displays:  
 "Login successful"

Otherwise, it shows:  
 "Invalid credentials"
